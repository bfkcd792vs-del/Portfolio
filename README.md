# Victor Feneis — Portfolio

Personal portfolio built with Next.js, presenting my background in industrial engineering, supply chain, and data analysis.

---

## Tech Stack

- Next.js 14 (App Router)
- React / TypeScript
- Framer Motion
- Tailwind CSS / shadcn/ui
- Spline (3D elements)
- Vercel (deployment)

---

## Sections

- **Hero** — introduction and contact links
- **Projects** — personal and academic projects
- **Experience** — internships and professional background
- **Skills** — tools and technologies
- **Contact** — contact form

---

## Local Setup

```bash
git clone https://github.com/bfkcd792vs-del/Portfolio.git
cd Portfolio
npm install
npm run dev
```

Open http://localhost:3000
